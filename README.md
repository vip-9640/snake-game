# snake-game

A simple browser Snake game built with plain HTML, CSS, and JavaScript.

## Features
- Desktop controls: arrow keys or WASD
- Mobile controls: swipe on the board or tap the on-screen arrows
- Pause/resume support (Space key or Pause button)
- Score, snake length, and local high score tracking
- Progressive speed increase as score grows

## Run locally
Open `index.html` directly, or serve the folder with a static server:

```bash
python3 -m http.server 8000
```

Then go to `http://localhost:8000`.
