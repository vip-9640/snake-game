# snake-game

A simple browser Snake game built with plain HTML, CSS, and JavaScript.

## Features
- Works on desktop with arrow keys or WASD
- Works on mobile with swipe controls and touch buttons
- Score + local high score tracking
- Restart button for quick replay

## Run locally
Open `index.html` directly, or serve the folder with a static server:

```bash
python3 -m http.server 8000
```

Then go to `http://localhost:8000`.
